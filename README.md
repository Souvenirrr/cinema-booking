# Ứng dụng đặt mua vé xem phim

## Idea
- Ứng dụng đặt vé xem phim cho rạp chiếu phim.
- Khách hàng mở ứng dụng -> chọn phim -> chọn lịch, địa điểm -> chọn ghế ngồi -> mua vé.

## Platform
- IOS, Android.
